# Консольное приложение для службы доставки

## Функции

- ввод информации о номере заказа, весе, районе заказа, времени доставки через консоль 
- фильтрация заказов в зависимости от количества обращений в конкретном районе города и времени обращения с и по.
- логирование операций
- валидация  входных данных
- вывод информации о всех заказах
- вывод результатов фильтрации заказов для доставки в конкретный район города в ближайшие полчаса после времени первого заказа в файл ({название области}/sortOrders.json)

## Особенности 

- данные о заказах хранятся в orders.json файле
- логи хранятся в logFile.txt файле
- 3 уровня логирования
- конфигурация хранится в файле App.config
- набор тестовых данных хранится в orders.json (при первом запуске выбрать в свойствах Copy if newer для копирования в папку сборки)

## Используемые библиотеки

- Newtonsoft.Json Version="13.0.3" для работы с Json файлами
- System.Configuration.ConfigurationManager Version="8.0.1" для работы с файлом конфигурации
